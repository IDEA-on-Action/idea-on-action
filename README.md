## 🚀 프로젝트 개요

**VIBE WORKING**은 AI 기술을 활용한 혁신적인 워킹 솔루션을 제공하는 플랫폼입니다. 우리는 개인과 기업이 더 효율적이고 창의적으로 일할 수 있도록 돕는 AI 기반 도구와 서비스를 개발합니다.

### 🎯 핵심 가치
- **혁신**: 최신 AI 기술을 활용한 솔루션 제공
- **효율성**: 업무 생산성 극대화
- **창의성**: 새로운 아이디어와 접근 방식 제시
- **신뢰성**: 안정적이고 검증된 기술

## 🛠️ 기술 스택

### Frontend
- **React 18** - 현대적인 UI 프레임워크
- **TypeScript** - 타입 안전성 보장
- **Vite** - 빠른 개발 환경
- **Tailwind CSS** - 유틸리티 우선 CSS
- **shadcn/ui** - 고품질 UI 컴포넌트

### Backend & 인프라
- **Supabase** - 백엔드 서비스 (인증, 데이터베이스)
- **Vercel** - 자동 배포 및 호스팅
- **GitHub Actions** - CI/CD 파이프라인

### 개발 도구
- **ESLint** - 코드 품질 관리
- **Prettier** - 코드 포맷팅
- **React Query** - 서버 상태 관리
- **React Router** - 클라이언트 사이드 라우팅

## 🚀 빠른 시작

### 개발 환경 설정
```bash
# 1. 저장소 클론
git clone https://github.com/IDEA-on-Action/IdeaonAction-Homepage.git
cd IdeaonAction-Homepage

# 2. 의존성 설치
npm install

# 3. 환경 변수 설정 (.env.local)
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# 4. 개발 서버 실행
npm run dev
```

### 주요 명령어
```bash
npm run dev       # 개발 서버 실행 (http://localhost:8081)
npm run build     # 프로덕션 빌드
npm run preview   # 빌드 미리보기
npm run lint      # 코드 품질 검사
```
